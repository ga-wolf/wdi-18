# Set up of Babel

- ` npm install --global webpack `
- ` npm init `
- ` npm install --save-dev webpack babel babel-core babel-loader babel-preset-es2015 `
- Sort out ` .gitignore `
- Configure babel with .babelrc
- Sort out webpack.config.js
  - entry
  - output
  - loaders
  - plugins
- Run ` webpack `
