DOM Readiness

In order to do cool jQuery stuff, we need to make sure that all of the content in the HTML document has been parsed by the browser before our script is run, so put all your DOM-related jQuery code in a function that is called when the `document` is `ready` - $(document).ready(function() { } );

Event Delegation

Event Listeners (like $("p").on("click", function() {});) are added to elements when the page is first loaded. This means that if we dynamically add elements to the page after the page is loaded, any Event Listeners set up in our JS will not be attached to those new elements.

One way to get around this is with Event Delegation. Here, we bind the event listener to an element that we know will be in the DOM when the page is loaded, and pass in an additional 'filter' argument to the event handler. Eg: $(".container").on("click", "p", function() { console.log("paragraph clicked")}) .
